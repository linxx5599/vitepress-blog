Vue

虚拟滚动条（scrollbar-vue2）

这是一个基于 vue2 的虚拟滚动条

安装

npm install scrollbar-vue2

pnpm add scrollbar-vue2

github 文档(参数说明) scrollbar-vue2

example

<template>
  <div class="tag">
    <Scrollbar :always="true" class="scrollbar">
      <div class="tags" style="height: 200px">
        <div v-for="t in 800" :key="t">{{ t }}</div>
      </div>
    </Scrollbar>
  </div>
</template>
<script>
import Scrollbar from "scrollbar-vue2";
export default {
  components: { Scrollbar }
}
</script>
<style lang="scss">
.tag {
  display: flex;
  align-items: center;
  justify-content: space-between;
  background-color: rgba(116, 116, 116, 0.281);
  height: 100px;
  .scrollbar {
    width: 100%;
    .tags {
      display: flex;
      width: 1px;
    }
  }
}
</style>

vue3个人组件库

基于 vue3 开发的个人组件库（yuan-design）采用pnpm workspace 管理项目 文档采用 vuepress
使用技术 ts + vue3 + vite + pnpm 详情请跳转 github yuan-design进行查看

git clone https://github.com/linxx5599/yuan-design.git
cd yuan-design
pnpm i
#组件预览
pnpm dev:dev
#文档预览
pnpm dev:docs
#打包组件用于发布
pnpm build:lib

俄罗斯方块

react18开发的俄罗斯方块小游戏 tetris-react18

git clone https://github.com/linxx5599/tetris-react18.git
cd tetris-react18
yarn
yarn dev

目前仅支持单机模式 联网对战搁置了 （koa + socket.io） 注意仅支持pc端，暂不支持移动端

chat 聊天室

react18 + node(express) + knex + mysql + socket.io 开发的聊天室 chat

git clone https://github.com/linxx5599/chat.git
# 启动前端
cd chat/web
yarn
yarn dev
# 启动服务端
cd chat/server
yarn
yarn start
